## ClickMonitorDDC Command Line Options

| Option  | Code | Value Range |
| ------------- | ------------- | -- |
| Brightness | b | 0-100 |
| Contrast | c | 0-100 |
| Red | r | 0-255 |
| Green | g | 0-255 |
| Blue | bl, l , i | 0-255 |


Example command setting brightness to 80, contrast to 74, red to 204, green to 214, blue to 242:
```
b 80 c 74 r 204 g 214 bl 242
```
